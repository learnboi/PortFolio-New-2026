# Fonts Directory

## Louize-Regular-205TF Font

Please add your `Louize-Regular-205TF` font file to this directory.

### Supported Formats (in order of preference):
1. `.woff2` (best compression, modern browsers)
2. `.woff` (good compression, wide support)
3. `.ttf` (TrueType, universal support)
4. `.otf` (OpenType, universal support)

### File Name
The font file should be named exactly: `Louize-Regular-205TF.woff2` (or `.woff`, `.ttf`, `.otf`)

### Usage
The font is automatically loaded via `@font-face` in `src/styles/index.css` and used for:
- Section titles
- Headings (h1, h2, h3)
- Brand name in navbar
- Hero section name

### Fallback
If the font file is not found, the site will fall back to 'Open Sans' font.
