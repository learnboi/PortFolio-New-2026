7. Deployment Guide

7.1 Current Deployment

  • Platform: Vercel
  • Domain: devchaitanya.online
  • Repository: https://github.com/learnboi/PortFolio-New-2026

7.2 Deployment Process

  1. Code pushed to GitHub (main branch)
  2. Vercel automatically detects changes
  3. Builds the project using npm run build
  4. Deploys to production
  5. Updates live site at devchaitanya.online

7.3 Build Configuration

Build Command: npm run build
Output Directory: build
Framework: Create React App
Install Command: npm install

7.4 Domain Configuration

  • Nameservers: ns1.vercel-dns.com, ns2.vercel-dns.com
  • SSL Certificate: Automatic (free)
  • DNS Propagation: 1-24 hours

8. Customization Guide

8.1 Updating Projects

Edit src/components/Projects.js and update the projects array with your GitHub repository URLs.

8.2 Changing Colors

Edit src/styles/index.css and modify CSS variables:

:root {
  --primary-color: #6366f1;
  --secondary-color: #8b5cf6;
}

8.3 Adding New Sections

  1. Create component in src/components/
  2. Create CSS file for styling
  3. Import in src/App.js
  4. Add to navigation in src/components/Navbar.js
